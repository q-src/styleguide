Eléments standard

Titre de page

FIX

Chaque page, à l’exception de la page d’accueil, doit comporter un titre de page. Celui-ci constitue toujours le premier élément de la page de contenu. Il n’est précédé du lien «Retour à l’aperçu» que sur les pages de détails non reliées (p. ex. pour les publications).

Le texte du titre de la page est pertinent pour le positionnement dans les résultats de recherche (p. ex. sur Google).

This is the page title, and it can be really quite long.

<h1 class="page-title">This is the page title, and it can be really quite long.</h1>

Texte et image

FLEX

Cet élément se compose d’un corps de texte pouvant être complété par un titre et des intertitres. Une image avec légende d’image, qui peut être habillée par le texte à droite , peut également être ajoutée. Diverses tailles d’images (principalement dans les proportions du nombre d’or) sont prévues.

Pour les petites images, on peut utiliser les classes pull-left ou pull-right dans l’élément HTML <figure>.

  • ajoutez une classe .clearfix au wrapper .well
