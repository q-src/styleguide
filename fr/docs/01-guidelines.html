Application des lignes directrices web de la Confédération

Structure

A l’intérieur des Lignes directrices pour le design web de la Confédération, vous pouvez naviguer vers les différents chapitres via les onglets du menu principal.

La page d’accueil présente un résumé des Lignes directrices pour le design web de la Confédération.

La structure, la compatibilité avec les navigateurs et l’installation sont décrites sous «Lignes directrices pour le design web de la Confédération».

Sous les onglets «Grille de base», «Modules de navigation» et «Modules de contenu», vous trouverez les descriptions détaillées des divers éléments.

Le menu «Pages d’exemple», présente différents types de pages Internet. Celles-ci sont ouvertes dans un nouvel onglet. Elles peuvent y être mises à l’échelle en vue de vérifier l’affichage dans différentes résolutions.

Informations sur les éléments

Sur les différentes pages, l’élément est affiché dans un cadre adaptatif. En agrandissant et en rétrécissant le cadre, on peut tester l’affichage de l’élément sur différents périphériques de sortie (design web adaptatif). En-dessus de l’affichage de l’élément, vous trouverez une brève description ainsi que le code pour la mise en œuvre par l’équipe de développement.

Le code peut être copié et réutilisé.

Les notes de mises à jour sont également indiquées. Celles-ci informent les développeurs à propos des changements de code.

La rubrique «Pages d’exemple» présente différents types de pages. Celles-ci sont adaptatives et accessibles.

Navigateurs web pris en charge

La version 3.0 est optimisée pour les versions de navigateur suivantes:

Ordinateur de bureau:

  • Internet Explorer 9 et supérieur
  • Safari 8 et supérieur
  • Firefox 40 et supérieur
  • Chrome 45 et supérieur

Mobile:

  • Safari et Chrome à partir d’iOS 7 et supérieur
  • Chrome et ou «default browser» et Firefox à partir d’Android 5 et supérieur

Installation

Les informations d’installation pour l’utilisation des lignes directrices dans le cadre d’un projet peuvent être consultées sur Github
